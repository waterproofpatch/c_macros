# C Macros

Some useful C macros.

## Prerequisites

* arm-none-eabi-gcc 
```bash
brew tap ArmMbed/homebrew-formulae
brew install arm-none-eabi-gcc`
```

## Build

```bash
make
```

## Run

```bash
./bin/main
src/main.c:11: [DEBUG] A debug message
src/main.c:12: [ERROR] An error message
src/main.c:13: [INFO] An info message
```
